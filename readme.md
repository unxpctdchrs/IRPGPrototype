## 🎮 Controls

### World Exploration
| Input | Action |
|-------|--------|
| `W A S D` | Move player |
| `E` | Interact |
| `Escape` | Pause |

### Battle — Character & Target Selection
| Input | Action |
|-------|--------|
| `A / D` | Cycle between characters / targets |
| `Enter` | Confirm selection |
| `Escape` | Cancel / go back |

### Battle — Action Menu
| Input | Action |
|-------|--------|
| `W / S` | Navigate actions (Attack, Skill, Item) |
| `Enter` | Confirm action |
